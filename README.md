# Baixada organization defaults

This repository holds the public profile and shared community health files for
the Baixada organization on GitHub.

GitHub uses these files as defaults for organization repositories that do not
provide a repository-specific version:

- `profile/README.md` is the organization profile.
- `CONTRIBUTING.md` describes the common contribution path.
- `CODE_OF_CONDUCT.md` sets the standard for project spaces.
- `SECURITY.md` defines private vulnerability reporting.
- `SUPPORT.md` routes questions and support requests.
- `.github/ISSUE_TEMPLATE/` provides structured issue forms.
- `.github/PULL_REQUEST_TEMPLATE.md` provides the shared pull request outline.

A repository may override any default when its audience, risk, or development
workflow needs more specific guidance. The `LICENSE` in this repository covers
these organization-default files; every other repository states its own
license.

The public organization profile is maintained in
[`profile/README.md`](profile/README.md).
