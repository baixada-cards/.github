# Support

Baixada is a volunteer project. Support is provided through GitHub when project
capacity permits, without a guaranteed response time.

Use the affected repository's:

- **Issues** for reproducible defects;
- **Discussions** for usage and design questions, when Discussions is enabled;
- **Question form** when Discussions is not available;
- **Security tab** for suspected vulnerabilities.

Before posting, search the repository's documentation, issues, and discussions.
Include the version or commit, environment, expected result, actual result, and
the smallest reproduction you can provide. Remove credentials, private game
data, and personal information.

Rules questions are welcome when they identify the game and variant. Baixada's
first Truco implementation is two-player *truco paulista*; a behavior from
another regional variant may be intentional rather than a defect.

Requests for deadlines, private consulting, account recovery, and guarantees
of compatibility are outside the project's support commitment.
